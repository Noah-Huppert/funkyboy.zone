Routine

  1. Back (Deadlift)
  2. Arms / Chest (Bench)
  3. Rest
  4. Legs / Core (Squat)
  5. Shoulders / Arms (Push Press)
  6. Rest / Optional Cardio
  7. Rest

Back

  • Hyper Extension (3 x 12-15, 30s rest)
  • Stationary Lunge (2 x 8-12, 30s rest)
  • Deadlift (5 x 5, 2-3m rest)
  • TBar Row (4 x 8-12, 1m rest)
  • Wide Grip Cable Row (4 x 8-12, 1m rest)
  • Wide Grip Lat Pulldown (4 x 8-12, 1m rest)
  • Smith Machine Shrugs (4 x 12-15, 1m rest)

Arms / Chest

  • Flat Barbell Press (5 x 5, 1m rest)
  • Dumbell Incline Press (3 x 8-12, 1m rest)
  • Hammer Strength Decline (3 x 8-12, 1m rest)
  • Pec Deck (3 x 12-15, 1m rest)
  • Dips (3 x 8-12, 1m rest)
  • Skillcrusher (3 x 8-12, 1m rest)
  • Dumbbell Kick Back (3 x 12-15, 1m rest)
  • Barbell Curl (3 x 6-8, 1-2m rest)
  • Hammer Curl (3 x 6-8, 1m rest)
  • Dumbbell Wrist Curl (2 x 20-25, 30s rest)

Legs / Core

  • Hyper Extension / Lunge (3 x 12-15, 30s rest)
  • Squat (5 x 5, 2-3m rest)
  • Leg Press (5 x 10, 1-2m rest)
  • Stiff Leg Deadlift (5 x 8-12, 1m rest)
  • Glute Bridge (5 x 12-15, 1m rest)
  • Seated / Standing Calf Raise (5 x 8-12, 1m rest)
  • Hanging Leg Raise (50, any rest)
  • Inclined Sit Ups On A Decline Bench (50, any rest)

Shoulders / Arms

  • Reverse Pec Dec (3 x 8-12, 1m rest)
  • Pull Ups (3 x 8-12, 1m rest)
  • Standing Push Press (5 x 5, 1m rest)
  • Seated Dumbbell Overhead Press (3 x 8-12, 1-2m rest)
  • Seated Dumbbell Lateral Raise (3 x 8-12, 1m rest)
  • Seated Dumbbell Front Raise (3 x 8-12, 1m rest)
  • V-Grip Press-down (4 x 8-12, 1m rest)
  • Rope Press-down (3 x 12-15, 1m rest)
  • Seated Dumbbell Preacher Curl (3 x 8-12, 1m rest)
  • Reverse Thumb-less Grip Barbell Curl (3 x 12-15, 1m rest)
  • Dumbbell Wrist Curl (2 x 20-25, 30s rest)
