Overview

Funky Boy provides a WireGuard VPN.

The VPN can provide access to the Funky Boy private network. Additionally it allows you to route personal internet traffic through Funky Boy.

Setup

  1. Install WireGuard
  2. Generate Keys:
    wg genkey > private.key
    chmod 600 private.key
    wg pubkey < private.key > public.key
  3. Contact the Funky
  4. Make a configuration file named wg0.conf:
    [Interface]
    Address = 10.0.0.2/24
    PrivateKey = YOUR PRIVATE KEY

    [Peer]
    PublicKey = {{ pillar.wireguard.public_key }}
    Endpoint = funkyboy.zone:{{ pillar.wireguard.interface.port }}
    AllowedIPs = 0.0.0.0/0
    PersistentKeepalive = 25
  5. Start WireGuard:
    wg-quick up wg0.conf
