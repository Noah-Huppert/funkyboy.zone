Download MSFS 2020 Plan File (2020-320-sim-pilot-community-holiday-flight.pln)

This Microsoft Flight Simulator 2020 plan file has all the details of our flight pre-configured. All you have to do is click the "LOAD/SAVE" button at the bottom of the flight planning screen, then click "LOAD" and select this file.
