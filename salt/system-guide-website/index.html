Funky Boy Guide

Overview

The Funky Boy Linux server is setup using a configuration management tool named Salt.

If you would like to install a package or edit a configuration file you must do so via Salt.

If you do not then there is a strong chance that the server will be re-setup with Salt, and you manual changes will be erased.

Instructions

Salt configuration files are stored in the Funky Boy GitHub repository.

See instructions in this repository for more information.
